Vinculos

Para editar el aspecto visual de nuestro vinculos debemos tener en cuenta que los mismos están compuestos por 4 estados:

  • Reposo (a:link)
  • Visitado (a:visited)
  • Hover (a:hover)
  • Activado (a:active)

Debemos respetar el orden de las pseudo-clases cuando escribamos nuestro CSS. No debe faltar ninguna.

Flexbox

Es un método de CSS para ordenar elementos en filas y/o columnas. Además, sirve para que los elementos sean flexibles dentro de un contenedor.
